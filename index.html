About Pac-Man

I started Pac-Man when I was ~10 as a Hatch Coding project and it just kind-of improved from there. The original version was really bad, but over time I made many improvements to it and now it is very well done. Then, during the summer of 2022, I converted the ProcessingJS to Processing 4.2(Java). Now, I have converted the Processing code to pure Java. I am working on an auto-update feature for the Windows version.
